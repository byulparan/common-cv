#### Require:
  - [OpenCV](http://opencv.org/)
  - [CvBlob](https://code.google.com/p/cvblob/) - optional
  - [Quicklisp](http://www.quicklisp.org)

#### Installation:
Mac OS X - I use homebrew: `brew install opencv`

Ubuntu - `sudo apt-get install libhighgui-dev`
